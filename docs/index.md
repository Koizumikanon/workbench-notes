# Workbench Notes

这里记录可以公开分享的技术实践：系统维护方法、工具使用经验、自动化工作流和工程决策。

所有文章都遵循一个边界：讲清楚方法，但不暴露真实环境。主机名、内网地址、账号、域名、密钥、日志和业务拓扑都应使用示例值或被移除。

## 当前内容

- [公开前审核](guides/publishing-safely.md)：什么内容不应公开，以及如何检查。
- [硬件维护预检](guides/hardware-maintenance-precheck.md)：把生产维护前的检查整理成可复用方法。

## 站点范围

本站不是某个公司环境的操作手册。它只发布可泛化、经脱敏审核的内容；真实 runbook 和内部资料应保留在私有仓库或受控系统中。
